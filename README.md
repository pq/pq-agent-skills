# 🚧 Agent Skills (WIP)

A collection of custom skills for agentic workflows.

(Limited to Gemini / Antigravity for now.) 

## Skills Included

### 📊 Estimate Conversation Tokens
Analyzes the full conversation transcript logs (`transcript_full.jsonl`) for the **current active conversation** (automatically resolved via environment variables) to estimate character lengths, model invocations (turns), cumulative input/output tokens, and context caching savings.

### 🏷️ Token Categories Explained

The script groups the conversation's raw characters and tool interactions into the following categories:

* **📄 File I/O (Reads/Edits):** Measures the token footprint of reading or writing files in the workspace (via `view_file` or `notebook_edit`).
* **🖥️ Command Execution:** Counts tokens generated by running terminal commands and capturing stdout/stderr (via `run_command`).
* **🔍 Search & Directory Listing:** Tracks tokens spent searching code patterns or listing directory trees (via `grep_search` or `list_dir`).
* **🌐 Web Search & Reads:** Captures tokens consumed during web search lookups or reading documentation links (via `search_web` or `read_url_content`).
* **📚 Knowledge Base Overhead:** Measures the size of automatically injected local Knowledge Items (KIs) at the beginning of the conversation.
* **🧠 Agent Reasoning & Output:** Reflects the model's generated response tokens, including its internal "thinking" process and tool call definitions.
* **💬 User Input & Other Metadata:** Counts raw user prompt text and miscellaneous system-generated messages/metadata.
* **⚙️ System & History Overhead:** Measures the cumulative cost of the core system prompt, agent tool definitions, and previous turn history re-sent on every loop.

## 🚀 Installation

### Option 1: Global Setup (Available everywhere)
Clone the repository and symlink the skill to your global config directory:
```bash
git clone https://github.com/pq/pq-agent-skills.git ~/src/pq-agent-skills
mkdir -p ~/.gemini/config/skills
ln -s ~/src/pq-agent-skills/skills/estimate-conversation-tokens ~/.gemini/config/skills/estimate-conversation-tokens
```

### Option 2: Workspace Integration (Project-specific)
Add this repository to your project's `.agents/skills.json` under your `entries` paths:
```json
{
  "entries": [
    { "path": "<path-to-checkout>/pq-agent-skills/skills" }
  ]
}
```
*Note: Make sure to replace `<path-to-checkout>` with the absolute local path to your checkout.*

## 🛠️ Usage / Invocation

Once the skill is installed, you can invoke it in your agent chat interface in two ways:

1. **Via Slash Command:**
   Type the slash command directly in the chat:
   ```text
   /estimate-conversation-tokens
   ```

2. **Via Natural Language:**
   Ask the agent directly:
   > *"Estimate token usage for this conversation."*
